기본 정보

  • 역할
    근거리/단일/딜러

  • 스탯
    HP : 20 | 쿨타임 : ? | 이동속도 : 0.1 | 공격속도 : 4 | 공격력 : 1

  • 능력
    4개중 랜덤으로 한개 발동해 구독자 수를 올린다
    구독자가 0명 미만일시 계정을 새로파며 1.6만명 이상이 되면 랜덤으로 한명을 공개저격을 하여 위더5를 5초동안 준다 데미지를 준다
    이후 구독자가 만명이 떨어진다
  • [능력1] 영상을 올린다 / 구독자 변화율 : 1000~3000 / 재사용 쿨타임 : 3초 / 확률 60%
  • [능력2] 합방을 한다 / 구독자 변화율 : 2000~3000 / 재사용 쿨타임 : 5초 / 확률 10%
  • [능력3] 논란이 된 단어를 사용 / 구독자 변화율 : -6000~-2000 / 재사용 쿨타임 : 7초 / 확률 : 20%
  • [능력4] 해킹을 당한다 / 구독자 변화율 -100~-10 / 발광효과 지속 시간 : 15초 / 재사용 쿨타임 : 15초 / 확률 : 10%'
