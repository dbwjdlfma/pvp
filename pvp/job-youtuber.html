유튜버

기본 정보

  • 역할
    근거리/단일/딜러

  • 스탯
    HP : 20 | 쿨타임 : ? | 이동속도 : 0.1 | 공격속도 : 4 | 공격력 : 1

  • 능력
    4개 중 랜덤으로 한 개 발동해 구독자 수를 올린다.
    구독자가 0명 미만일 시 계정을 새로 파며, 1.6만 명 이상이 되면 랜덤으로 한 명을 공개저격하여 해당 플레이어에게 8초간 위더5, 3초간 구속3 효과를 준다.
    이후 구독자가 1.5만 명 감소한다.
  • [능력1] 영상을 올린다 / 구독자 변화율 : 100~3000 / 재사용 쿨타임 : 5초 / 확률 55%
  • [능력2] 합방을 한다 / 구독자 변화율 : 3000~5000 / 재사용 쿨타임 : 10초 / 확률 10%
  • [능력3] 논란이 된 단어를 사용 / 구독자 변화율 : -6000~-3500 / 재사용 쿨타임 : 7초 / 확률 : 25%
  • [능력4] 해킹을 당해 현재 위치가 채팅으로 노출이 된다 / 구독자 변화율 -100~-10 / 재사용 쿨타임 : 15초 / 확률 : 20%
