로봇

역할

원거리,근거리/광역/딜러


스탯

HP : 25 | 쿨타임 : 10s | 이동속도 : 0.1 | 공격속도 : 0 | 공격력 : 0


레벨 1

로봇 위로 3칸 이내에 블록이 있으면 [레벨 1] 상태가 됩니다.
이동속도는 기존과 동일하며 점프와 공격은 불가능합니다.
3m 이내에 다른 플레이어가 있으면 0.1의 대미지를 지속적으로 줍니다.
위로 3칸 이내에 블록이 3초 동안 없으면 [레벨 2]로 돌아갑니다.


레벨 2

점프키를 꾹 누르면 바라본 방향으로 날아갑니다.
날아가는 위력 = 점프키를 누른 시간(0.1초~) X 0.1 X 0.1
착지할 때 5m 이내에 있는 플레이어에게 0.1초간 구속을 줍니다.
착지할 때 (떨어진 틱 X 0.2)만큼 대미지를 줍니다.
